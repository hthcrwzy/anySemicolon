## get_char_code.go
### Features
- Output decimal character codes.  
    Example:
    ```console
    $ go run get_char_code.go anySemicolon ";"
    This program outputs decimal character codes.

    a: 97
    n: 110
    y: 121
    S: 83
    e: 101
    m: 109
    i: 105
    c: 99
    o: 111
    l: 108
    o: 111
    n: 110
    ;: 59
    ```
